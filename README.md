# lessampler
 lessampler is a Singing Voice Synthesizer

## 目标：
基于JUCE的歌声合成软件，提供高质量，高效率的歌声合成体验。

## API:
- [ ] [UTAU](http://utau2008.xrea.jp/) 接口
- [ ] [LeafOpenUTAUQt](https://github.com/shine5402/LeafOpenUTAUQt) 接口
- [ ] Library 接口

- [ ] 错误提示对话框
- [ ] 信息提示对话框
- [ ] 警告提示对话框
- [ ] GUI引擎参数修改器
- [ ] 版权窗口
