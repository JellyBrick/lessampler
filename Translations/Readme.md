### Structure of LANGPACK:

```json
{
 "LC" : "location",
 "LESSAMPLERLANG" :
  {
      "Key" : "Value",
   }
}
```

### Example:

``` json
{
	"LC" : "en_US",
	"LESSAMPLERLANG" : 
	{
		"Str1" : "The input parameter is wrong. please check if it is called correctly or the parameter is correct.",
		"Str2" : "Use Suggestions",
		"Str3" : "Please call in the UTAU program or call with relevant parameters",
		"Str4" : "Whats Wrong???",
		"Str5" : "Whats Wrong??? Should not be like this......"
	}
}
```
